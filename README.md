# Scoop bucket for personal purpose (j2doll)

## How to apply bucket

- Add bucket

```cmd
scoop bucket add j2doll-bucket https://github.com/j2doll/j2doll-scoop-bucket.git
```

- List bucket

```cmd
scoop bucket list
scoop search hello
```

- Update bucket

```cmd
scoop update
```

- Remove bucket

```cmd
scoop bucket rm j2doll-bucket
```
